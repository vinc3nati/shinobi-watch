## Features

- Sleek Landing Page
- Progressive Web App
- Login and Sign up with [MockBee](http://mockbee.netlify.app/)
- Genre filters
- Filter by latest, old uploads
- Search for videos
- Private Routes
- User Profile Page
- Playlist
- Watch Later
- History
- Liked videos
- Upload your own YouTube videos

- Wait for it to be reviewed
